# Qiadat - Oman Volunteer Center Platform

A modern, responsive web platform for the Oman Volunteer Center (Qiadat), built with **Next.js 14**, **Tailwind CSS**, and **TypeScript**.

## Features

- **Visual Identity**: Custom Navy (#1F2A52) and Cream (#EFE8BA) color scheme.
- **Home Page**:
  - Hero Section with overlay.
  - Quick Access Grid (overlapping design) for key services.
  - News Feed and Upcoming Events.
- **Services Portal**:
  - Hall Booking with date/time validation (Sun-Thu, 08:30-19:30).
  - Volunteer Request with dynamic "Working Days" notice period logic.
  - Event Enrollment.
  - Initiative Request (Gated).
- **Core Pages**:
  - About Us (Vision, Mission, Values).
  - Objectives (with Watermark).
  - Media Center (Tabs).
  - Partners & Contact.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser.

3. **Visit https://qiadat-liart.vercel.app/ to view the site**
